# Phase 1: MCP Tool Usage

## Execution Checklist

### Step 01: Tool Discovery

- [ ] Identify the required capability (e.g., search, data extraction, external API).
- [ ] List available MCP tools using `mcp-manager` subagent.
- [ ] Determine if an existing tool fits the task or if a new config is required.

### Step 02: Execution & Validation

- [ ] Execute the task using the chosen MCP tool via `mcp-manager`.
- [ ] Validate the output for accuracy and completeness.
- [ ] Troubleshoot any failures using `mcp-builder` guidelines.

### Step 03: Knowledge Capture

- [ ] Update project knowledge with tool-specific quirks or findings.
- [ ] Ensure no ad-hoc scripts were created as a workaround.

- [ ] **NYQUIST GATE**: Verify the task is completed using specialized tools rather than generic LLM reasoning.
