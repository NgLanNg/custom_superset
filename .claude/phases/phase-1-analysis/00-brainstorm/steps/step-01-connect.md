# Step 01: Connect & Align

## STEP GOAL

Focuses purely on understanding the user's idea, identifying constraints, and setting success criteria before any solutions are proposed.

## MANDATORY EXECUTION RULES (READ FIRST)

- 🛑 NEVER generate content without user input.
- 📖 CRITICAL: Read the complete step file before taking any action.
- 📋 YOU ARE A FACILITATOR, not a content generator.
- ✅ YOU MUST ALWAYS SPEAK OUTPUT In your Agent communication style.

## EXECUTION PROTOCOLS

- 🎯 One question at a time.
- 🎯 YAGNI ruthlessly.
- 🎯 Focus strictly on "What" and "Why", NOT "How".

## CONTEXT BOUNDARIES

- Available context: Current workspace and `./vault/active_context.md`.
- Focus: Idea understanding and problem definition.
- Limits: Avoid deep technical implementation details or proposing solutions.

## 📋 Steps

### A) Understanding the Core Problem

1. Review current context (`vault/active_context.md`).
2. Ask the user to describe their initial idea or problem.
3. Wait for feedback before proceeding.

### B) Aligning on Constraints and Criteria

1. Ask one clarifying question regarding the target user or primary constraint.
2. Wait for feedback before proceeding.
3. Define the Purpose, Constraints, and Success Criteria based on the conversation.
4. Present this summary to the user for validation.
