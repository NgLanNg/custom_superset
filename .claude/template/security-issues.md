### 2. SQL Injection Vulnerability

**Severity:** CRITICAL
**Location:** `src/api/users.py` (line 45)

**Issue:**

```python
# Vulnerable to SQL injection
query = f"SELECT * FROM users WHERE username = '{username}'"
result = db.execute(query)
```

**Attack Example:**

```python
username = "admin' OR '1'='1"
# Results in: SELECT * FROM users WHERE username = 'admin' OR '1'='1'
# Returns all users
```

**Recommendation:**

- Use parameterized queries or ORM
- **Never** concatenate user input into SQL

**Fix:**

```python
# Safe version
query = "SELECT * FROM users WHERE username = ?"
result = db.execute(query, (username,))
```

**References:**

- OWASP: [A03:2021 - Injection](https://owasp.org/Top10/A03_2021-Injection/)
- CWE-89: SQL Injection

### 2. Weak Password Requirements

**Severity:** HIGH
**Location:** `src/auth/validators.py`

**Issue:**

```python
# Too permissive
def validate_password(password):
 return len(password) >= 6 # Only checks length
```

**Recommendation:**

```python
def validate_password(password):
 return (
 len(password) >= 12 and
 re.search(r'[A-Z]', password) and # Uppercase
 re.search(r'[a-z]', password) and # Lowercase
 re.search(r'[0-9]', password) and # Number
 re.search(r'[^A-Za-z0-9]', password) # Special char
 )
```

### 3. Insecure Direct Object References (IDOR)

**Severity:** HIGH
**Location:** `src/api/documents.py`

**Issue:**

```python
# No authorization check
@app.route('/api/documents/<doc_id>')
def get_document(doc_id):
 doc = Document.query.get(doc_id)
 return jsonify(doc) # Any user can access any document
```

**Recommendation:**

```python
@app.route('/api/documents/<doc_id>')
@require_auth
def get_document(doc_id):
 doc = Document.query.get(doc_id)
 # Check ownership
 if doc.user_id != current_user.id:
 abort(403)
 return jsonify(doc)
```

**References:**

- OWASP: [A01:2021 - Broken Access Control](https://owasp.org/Top10/A01_2021-Broken_Access_Control/)
- CWE-639: Authorization Bypass Through User-Controlled Key

### 2. Verbose Error Messages

**Severity:** MEDIUM
**Location:** `src/api/error_handlers.py`

**Issue:**


**Recommendation:**

```python

except Exception as e:
```

### 2. Weak TLS Configuration

**Severity:** MEDIUM
**Location:** Web server config

**Issue:** Allowing TLS 1.0/1.1 (deprecated)

**Recommendation:**

- Require TLS 1.2 minimum (prefer TLS 1.3)
- Use strong cipher suites
- Enable HSTS

**nginx example:**

```nginx
ssl_protocols TLSv1.2 TLSv1.3;
ssl_ciphers 'ECDHE-ECDSA-AES128-GCM-SHA256:ECDHE-RSA-AES128-GCM-SHA256';
add_header Strict-Transport-Security "max-age=31536000; includeSubDomains" always;
```
