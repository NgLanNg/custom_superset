# Step 04: Log Rotation

**Purpose:** Consolidate or archive large log files.

## Instructions

1. Identify large log files (>1MB or >1000 lines)
2. Archive logs older than 7 days
3. Compress archived logs
4. Maintain current log files for active debugging

## Output

- `rotated_logs`: List of archived log files
- `log_summary`: Size reduction report
