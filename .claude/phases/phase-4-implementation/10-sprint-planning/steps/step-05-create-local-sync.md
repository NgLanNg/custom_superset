# Step 05: Create Local Sync

**Purpose:** Generate local tracking file for agents.

## Instructions

1. Create `docs/sprint_status.yaml` or `docs/sprint_status.md`
2. Include all allocated stories with their Jira ticket IDs
3. Mark status as "ready-for-dev"
4. Include sprint dates and team capacity

## Output

- `sprint_status_path`: Path to the local tracking file
- `sprint_summary`: Human-readable summary of the sprint
